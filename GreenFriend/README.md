# Green Friend (iOS MVP)

Приложение для ухода за комнатными растениями на SwiftUI.

## Что есть сейчас

- Каталог растений (SwiftData)
- Виртуальный подоконник
- История и отметка полива
- Локальные напоминания о поливе
- Виджет "Полив сегодня"

## Основные экраны

- `Каталог`
- `Подоконник`
- `Настройки`

## Стек

- SwiftUI
- SwiftData
- UserNotifications
- WidgetKit

## Запуск

1. Открой `GreenFriend.xcodeproj`.
2. Выбери target `GreenFriend`.
3. Запусти на устройстве или симуляторе.

## Виджет

Убедись, что App target и Widget target используют один и тот же App Group в `Shared/AppGroup.swift`.
