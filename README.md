## Описание

**Green Friend** — офлайн-ориентированное приложение для ухода за комнатными растениями.
Основной сценарий: пользователь добавляет растения на виртуальный подоконник, отмечает полив, отслеживает просроченные задачи и получает напоминания.

Поиск работает по локальному каталогу, а при отсутствии совпадений выполняется fallback-поиск через **World Flora Online (WFO)**.

---

## Возможности

- **Подоконник (главный экран)**:
  - сортировка по ближайшей дате полива (просроченные сверху);
  - фильтр `Сегодня` (включает и просроченные);
  - быстрые действия свайпами: отметить полив / удалить растение;
  - визуальные индикаторы света и полива.
- **Карточка растения**:
  - фото-карусель (до 5 фото), выбор главного фото;
  - удаление фото с подтверждением;
  - автоматическая подгрузка фото из интернета, если фото удалены;
  - редактирование даты последнего полива через встроенный календарь.
- **Поиск растений**:
  - локальная база популярных комнатных растений;
  - fallback в WFO при отсутствии записи в локальном каталоге;
  - добавление на подоконник напрямую из результатов.
- **Напоминания о поливе**:
  - включение/отключение уведомлений;
  - настройка интервала повтора;
  - настройка времени старта напоминаний.
- **Виджеты WidgetKit**:
  - отображение статуса полива на сегодня;
  - синхронизация данных с основным приложением через App Group.
- **Темы**:
  - светлая и темная темы интерфейса;
  - переключение иконки приложения под тему.

## Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/DenisSmirnov-ios/FlowerHelper.git
   cd FlowerHelper
   ```

2. Откройте проект в Xcode:
   ```bash
   open GreenFriend.xcodeproj
   ```

3. Выберите схему `GreenFriend` и запустите приложение на устройстве или симуляторе.

## Данные растений

- Основной источник: локальный JSON-каталог в приложении.
- Дополнительный источник при поиске: **World Flora Online (WFO)**.
- Фото растений кэшируются локально после загрузки.
